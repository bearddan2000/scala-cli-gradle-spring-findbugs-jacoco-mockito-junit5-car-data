# scala-cli-gradle-spring-findbugs-jacoco-mockito-junit5-car-data

## Description
A POC for spring app using junit5,
findbugs, jacoco, mockito framework.

## Tech stack
- scala
- gradle
  - spring
  - junit5  
  - findbugs
  - jacoco
  - mockito

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
